# 🎬 Netflix Data Analysis System

## 📌 Project Overview

The Netflix Data Analysis System is an interactive analytics and recommendation platform built using Python and Streamlit. The project analyzes Netflix content trends, user preferences, genre popularity, and rating distributions to generate meaningful business insights and personalized movie recommendations.

The system processes Netflix dataset information including:

* Content Type
* Genres
* Ratings
* Release Year
* Duration
* Popularity Trends

---

## 🚀 Features

* Interactive Streamlit Dashboard
* Netflix Content Trend Analysis
* Genre Popularity Visualization
* Rating Distribution Analysis
* Movie Recommendation System
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)

---

## 🛠️ Tech Stack

* Python

* Matplotlib
